При первой попытке печати документа в веб-клиенте программа предлагает создать рабочий каталог для хранения временных файлов печати.

Вызывается по команде Настройка рабочего каталога печати в персональных настройках программы на странице Печать.

С помощью кнопки  выберите путь к папке на компьютере. Нажмите ОК.
