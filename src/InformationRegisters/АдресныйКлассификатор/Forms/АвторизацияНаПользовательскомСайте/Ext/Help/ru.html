Для загрузки адресного классификатора с сайта "1С" необходимо ввести данные регистрации на сайте поддержки пользователей 1С:Предприятия:

  • Введите Логин;
  • Введите Пароль.
  • Для того чтобы больше не вводить эти данные в дальнейшем, включите флажок Запомнить пароль.
  • Если вы еще не зарегистрированы, то с помощью гиперссылки можно Перейти к регистрации на сайте поддержки пользователей.
  • Нажмите Продолжить для перехода к загрузке адресного классификатора.
