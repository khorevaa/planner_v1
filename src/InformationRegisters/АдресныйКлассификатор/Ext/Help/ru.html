В адресном классификаторе хранятся адресные сведения, необходимые для работы в программе.

Для обеспечения единства структуры и состава адресной информации ГНИВЦ ФНС РФ выпускает классификатор адресных объектов (КЛАДР). В КЛАДР содержатся почтовые адреса РФ. Использование КЛАДР помогает заполнять адреса правильно и быстро. Без этого невозможна сдача отчетности. Например, при сдаче отчетности по работникам в налоговую инспекцию, в пенсионный фонд необходимо заполнить адреса по стандарту, иначе отчетность не будет принята. Кроме того, адреса могут меняться и не всегда можно уследить за этими изменениями самостоятельно. Программа позволяет управлять адресами и поддерживать их в актуальном состоянии с помощью КЛАДР.

Для всех регионов РФ имеется возможность загрузки адресных сведений с сайта поддержки пользователей программы 1С:Предприятие фирмы "1С", с диска информационно-технологического сопровождения (1С:ИТС), а также из файлов, распространяемых ГНИВЦ ФНС РФ.

Окно для работы с адресным классификатором открывается по команде Адресный классификатор раздела Администрирование - Поддержка и Обслуживание - Классификаторы или раздела Справочники.

Выводится сводная информация об имеющихся в программе адресных сведениях (информационное сообщение Загружено регионов о количестве регионов, по которым загружены адресные сведения), а также доступны основные операции по управлению ими: загрузка, очистка и проверка обновлений через Интернет.

Загрузка адресного классификатора

  • Нажмите Загрузить.
  • При загрузке классификатора можно выбрать один, несколько или все регионы России, по которым необходимо загрузить адресные сведения. Это удобно, т.к. можно не загружать адреса тех регионов, которые заведомо не будут использоваться в работе. Включите флажки рядом с наименованиями нужных регионов. Незагруженные регионы неактивны, выделены серым цветом. Для того чтобы выбрать все регионы для загрузки, нажмите , для того чтобы отменить выбор, нажмите кнопку . Для поиска регионов в списке используйте кнопку Найти. В нижней части списка идет подсчет количества выбранных регионов.
  • Гиперссылка в нижней части окна указывает, из какого источника будет происходить загрузка. По умолчанию гиперссылка принимает значение С сайта поддержки пользователей 1С:Предприятия.
    • Для загрузки необходимо заполнить данные для авторизации на сайте поддержки пользователей.
  • Для изменения источника загрузки классификатора перейдите по гиперссылке. Загрузка также возможна:
    • Из каталога на диске.
      • укажите путь к каталогу;
    • С диска 1С:ИТС.
      • укажите путь к диску.
  • После указания всех параметров нажмите Загрузить.
  • Во время загрузки программой выводится сообщение о состоянии загрузки:
    • количество загружаемых регионов;
    • источник загрузки;
    • загружаемый в текущий момент регион.
  • После успешной загрузки выводится сообщение. Нажмите Закрыть для возврата в программу.
  • При повторной загрузке адресных сведений, ранее загруженных в классификатор, они перезаписываются.
  • В веб-клиенте загрузка адресного классификатора не поддерживается.

Проверка обновления

  • Программа хранит версию последнего загруженного КЛАДР и с помощью команды Проверить обновление позволяет проверить наличие обновления КЛАДР на сайте фирмы "1С". Проверка производится независимо от источника, из которого классификатор был загружен ранее.
  • Если данные устарели, программа предлагает обновить версию адресного классификатора, а также программа указывает, по каким регионам доступно обновление.
  • Для перехода к загрузке классификатора нажмите Обновить. Рядом с регионами, которые нужно обновить, флажки проставляются автоматически.

Очистка адресного классификатора

  • Нажмите Очистить, для того чтобы удалить адресные сведения (например, ошибочно загруженные).
  • Включите флажки рядом с наименованиями нужных регионов. Незагруженные регионы неактивны, выделены серым цветом.
  • Для того чтобы выбрать для очистки все регионы, нажмите , для того чтобы отменить выбор, нажмите .
  • Для поиска регионов в списке используйте кнопку Найти. В нижней части списка идет подсчет количества выбранных регионов.
  • После установки необходимых параметров нажмите Очистить.
