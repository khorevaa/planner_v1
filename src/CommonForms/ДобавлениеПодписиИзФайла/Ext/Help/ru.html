Предназначена для добавления подписи к файлу. Эту возможность можно использовать, например, если подпись (файл подписи) была получена позже, чем подписанный файл.

Открывается из списка файлов (списка присоединенных файлов объекта), списка Присоединенные файлы или из карточки файла (карточки присоединенного файла) по команде ЭП и шифрование - Добавить ЭП из файла. Также можно найти эту команду в меню Все действия.

В списке выводится:

  • Файл ЭП - путь к файлу ЭП.
  • Комментарий - необходимая информация о файле ЭП.

Добавление ЭП из файла

Выберите один или несколько файлов с электронными подписями (ЭП).

  • Нажмите  Добавить,  с помощью кнопки  укажите путь к файлу подписи на жестком или сетевом диске.
  • Напишите Комментарий к подписи.

Удаление ЭП

  • Нажмите , для того чтобы Удалить файл ЭП из списка.
