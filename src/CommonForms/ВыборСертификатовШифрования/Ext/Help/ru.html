Предназначен для выбора сертификатов электронной подписи (ЭП) при шифровании файла.

Открывается из списка файлов (присоединенных файлов объекта), списка присоединенных файлов или карточки файла (карточки присоединенного файла) по команде ЭП и шифрование - Зашифровать. Также можно найти эту команду в меню Все действия.

В списке выводится:

  • Выбран - флажок;
  • Кому выдан - пользователь - владелец электронной подписи;
  • Кем выдан - организация, выдавшая ЭП;
  • Действителен до - дата действия сертификата ЭП;
  • Назначение - цель использования сертификата ЭП, например, "Защищенная электронная почта", "Проверка подлинности сервера". Для того чтобы просмотреть подробную информацию о назначении сертификата, наведите мышку на нужную ячейку.

Шифрование файла

  • Могут использоваться несколько сертификатов, например, если необходимо зашифровать файл для нескольких пользователей.
  • Перед шифрованием необходимо включить флажки у сертификатов пользователей, для которых файл будет зашифрован.
  • Строки с выбранными сертификатами становятся недоступными, выключить флажки уже невозможно при шифровании всех файлов списка.
  • Для подтверждения выбора нажмите Зашифровать.
  • Если у файла несколько версий, будут зашифрованы все версии, для шифрования потребуется некоторое время.
    • Перед шифрованием в этом случае программа выводит сообщение. Для подтверждения нажмите Да.  
  • Использованные для шифрования сертификаты указываются в карточке файла на вкладке Зашифрован для. Команда Зашифровать после этого становится неактивной.
