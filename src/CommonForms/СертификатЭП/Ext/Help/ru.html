Предназначена для просмотра основных свойств сертификата электронной подписи (ЭП).

Электронная подпись - реквизит любого электронного документа, с помощью которого можно удостовериться в неизменности электронного документа с момента его подписи. Для подписи необходимо иметь сертификат ключа ЭП. Контроль неизменности электронного документа осуществляется в результате проверки соответствия ЭП документа и сертификата ключа ЭП. Если ЭП соответствует сертификату, это является гарантией того, что документ не был изменен кем-либо с момента его подписания. Контроль неизменности электронного документа можно выполнить в любой момент времени.

Открывается с помощью кнопки Открыть сертификат (также можно воспользоваться двойным щелчком мыши) в списке сертификатов на вкладке Разрешено расшифровывать карточки файла (карточки присоединенного файла). Также можно открыть сертификат с помощью соответствующей команды из окна Подпись ЭП.

Основные сведения о сертификате ЭП

  • Кому выдан - личные данные владельца ЭП;
  • Кем выдан - данные об Удостоверяющем центре, который выдал ЭП;
  • Действителен до - дата, до которой действует ЭП;
  • Назначение - перечень назначений сертификата.

Выгрузка сертификата

Нажмите Выгрузить и укажите папку, для того чтобы сохранить сертификат на компьютер.

Проверка сертификата

Нажмите Проверить, для того чтобы проверить действительность сертификата.
