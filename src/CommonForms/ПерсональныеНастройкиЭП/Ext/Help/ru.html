Предназначена для настройки работы с ЭП.

Открывается из персональных настроек программы (вкладка Работа с файлами) по ссылке Настройка ЭП.

Персональные настройки электронной подписи (ЭП)

  • С помощью переключателя Действия при сохранении с ЭП задайте поведение программы при выполнении команды Сохранить с ЭП:
    • Сохранять все подписи - все подписи сохраняются;
    • Спрашивать - выбор сохраняемых подписей осуществляется из списка.
  • Задайте Расширение для зашифрованных файлов при отправке по почте, при выполнении команды Сохранить как и при экспорте папки (по умолчанию .p7m).
  • Напишите Расширение для файлов подписи (по умолчанию .p7s).
  • Напишите Путь модуля криптографии (для веб-клиента под управлением Linux) для операций с файлами (расшифровка, подписание, проверка подписи, просмотр сертификата).
  •  Выберите из списка Личный сертификат для шифрования - персональный сертификат электронной подписи (ЭП) для шифрования данных. При выборе сертификатов для шифрования данный сертификат будет всегда отмечен. Это нужно для того, чтобы нельзя было зашифровать данные без возможности их расшифровки.

Настройки криптографии

Перейдите по ссылке Настройки криптографии, для того чтобы настроить ЭП и шифрование.
