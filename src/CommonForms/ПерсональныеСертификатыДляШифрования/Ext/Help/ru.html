Предназначен для выбора персонального сертификата электронной подписи (ЭП), который будет применяться по умолчанию для шифрования данных.

Открывается по кнопке  Выбрать из персональных настроек ЭП.

Список состоит из колонок:

  • Кому выдан - данные о пользователе, который получил сертификат ЭП.
  • Кем выдан - данные об удостоверяющем центре (УЦ), выдавшем сертификат ЭП.
  • Действителен до - дата, по которую можно использовать данный сертификат ЭП.

Выбор сертификата

  • Выделите в списке нужный сертификат, нажмите Выбрать. Персональный сертификат будет отмечен при выборе сертификатов для шифрования.

Просмотр сертификата

  • Выделите нужный сертификат, нажмите Открыть.
