Предназначена для ввода пароля при расшифровке файла.

Открывается по команде ЭП и шифрование - Расшифровать (Все действия - ЭП и шифрование - Расшифровать) в списке файлов (присоединенных файлов), списке присоединенных файлов или карточке файла (карточке присоединенного файла).

Выводятся сведения:

  • В поле Файл указывается наименование файла, недоступно для редактирования.
  • В поле Файл зашифрован для указывается, для каких персональных сертификатов электронной подписи (ЭП) зашифрован файл, недоступно для редактирования.

    Расшифровка файла

    • В поле Пароль введите пароль для расшифровки файла, для подтверждения нажмите ОК.
