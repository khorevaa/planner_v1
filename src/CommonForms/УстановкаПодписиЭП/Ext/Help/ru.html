Предназначена для выбора сертификата электронной подписи (ЭП) при подписании данных.

Открывается по команде  Подписать, например, из списка файлов (присоединенных файлов).

Электронная подпись - реквизит любого электронного документа, с помощью которого можно удостовериться в неизменности электронного документа с момента его подписи. Для подписи необходимо иметь сертификат ключа ЭП. Контроль неизменности электронного документа осуществляется в результате проверки соответствия ЭП документа и сертификата ключа ЭП. Если ЭП соответствует сертификату, это является гарантией того, что документ не был изменен кем-либо с момента его подписания. Контроль неизменности электронного документа можно выполнить в любой момент времени.

Подписание данных

  • Подписывать файл могут разные пользователи, при этом для каждого будет создана своя подпись.
  • Для подписания выберите один сертификат.
  • Напишите Пароль к закрытому ключу (вводится в процессе создания сертификата).
  • При необходимости можно ввести Комментарий к подписи - дополнительную информацию о подписи. 
  • Для выполнения подписи нажмите Подписать.
  • Данные будут подписаны с использованием указанного сертификата ЭП.
  • Созданная ЭП хранится в программе. Подписанный файл становится недоступным для редактирования и отображается в списках со значком .
