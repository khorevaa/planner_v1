Предназначено для информирования об ошибке во время обновления ИБ.

Для того чтобы посмотреть в журнале регистрации сведения об ошибках, перейдите по гиперссылке Подробности см. в Журнале регистрации. 

Для того чтобы выйти из программы, нажмите Завершить.

Для того чтобы попытаться запустить программу еще раз, нажмите Перезапустить.
